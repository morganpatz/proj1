What this example do?

This example checks whether the user entered id and password pair is valid. The valid id and password pairs are stored in a table called login. If the id and password match the tuple in the underlying table, a success message will be displayed. If the id and password doesn't match any tuple in the underlying table, a failure message will be displayed.

Three Files in this example

There are 3 files in this example.
  • login.html is the user login page.
  • login.jsp is the jsp file checking whether the user input is valid.
  • login.sql is the sql file which create the table,login, and insert id/password pairs into that table.

    drop table login;

    create table login(id char(20),pwd char(20));

    insert into login values(john, 11111);

    insert into login values(mary, 22222);

    Install this example

    To execute this example successfully, you need to do the following:
  • execute login.sql to create the id/password table (login table)
  • copy login.jsp and login.html into your your_home_directory/catalina/webapps/proj1/
  • change the two string "your-sql-id" and "your-sql-password" to be your sql account name and sql password.
  • then start your tomcat
  • Go to http://machine_name:your_port_number/proj1/login.html
