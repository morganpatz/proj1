The following can be used to insert an image of BLOB type into an LogicSQL database.

Note that (1) the table in the database is created by 
      CREATE TABLE pictures (
            pic_id int,
	    pic_desc  varchar(100),
	    pic  BLOB,
	    primary key(pic_id)
      )
(2) a unique constraint violation will be returned if the table already contains a row with the same primary key value.
